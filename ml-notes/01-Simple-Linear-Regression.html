What is Simple Linear Regression?
Simple Linear Regression is finding the best relationship between the input variable x (independent variable)
and the expected variable y (dependent variable). The linear relationship between these two variables can be
represented by a straight line called regression line.

Estimating ("Learning") Model Coefficients
The coefficients are estimated using the least-squares criterion, i.e., the best fit line has to be calculated
that minimizes the sum of squared residuals (or "sum of squared errors").
